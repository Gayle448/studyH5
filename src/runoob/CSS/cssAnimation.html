一些css属性是可以有动画效果的，动画属性可以从尺寸大小、数量、百分比和颜色着手

background 属性是 *所有背景属性* 的简写属性。

CSS 中仅有 background-color, background-position, 和 background-size 属性支持动画效果。

注意：Internet Explorer 9 及更早 IE 版本不支持 CSS 动画。
