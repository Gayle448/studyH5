一些css属性是可以有动画效果的，动画属性可以从尺寸大小、数量、百分比和颜色着手

background 属性是 *所有背景属性* 的简写属性。

CSS 中仅有 background-color, background-position, 和 background-size 属性支持动画效果。

注意：Internet Explorer 9 及更早 IE 版本不支持 CSS 动画。

@keyframes 理解: 使用@keyframes规则，你可以创建动画。 创建动画是通过逐步改变从一个CSS样式设定到另一个。 在动画过程中，您可以更改CSS样式的设定多次。 指定的变化时发生时使用％，或关键字"from"和"to"，这是和0％到100％相同。 0％是开头动画，100％是当动画完成。 元素：动画名称、动画进度节点、动画样式 三个必需的

animation 动画：在css里面使用简写属性把 animation 绑定到一个div 元素； animation: name (动画名) duration （动画时长） timing-function （动画速度可以使用 速度曲线函数） delay （动画延迟） iteration-count (播放次数 infinite是无数次) direction (正向 反向播放方式) fill-mode (动画开始和结束的时候显示的样式) play-state (指定暂停时/运行时的动画); 这些值也是一个单独的属性。
