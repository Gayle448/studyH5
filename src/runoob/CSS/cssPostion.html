元素可以使用的顶部，底部，左侧和右侧属性定位。然而，这些属性无法工作，除非是先设定position属性。

static : 默认值;不会受到 top, bottom, left, right影响;

fixed : 元素的位置相对于浏览器窗口是固定位置。即使窗口是滚动的它也不会移动. Fixed定位使元素的位置与文档流无关，因此不占据空间。Fixed定位的元素和其他元素重叠。

relative : 相对定位，是指相对自己本应该的正常位置，做出的修改. 移动相对定位元素，但它原本所占的空间不会改变。

absolute : 绝对定位的元素的位置相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于html标签的. 某个标签用了绝对定位，它后面的标签不会跟着这个走，受这个标签影响了，从上到下正常排铺。 absolute 定位使元素的位置与文档流无关，因此不占据空间。 absolute 定位的元素和其他元素重叠。

sticky 粘性定位，设定一个阀值，改变则固定在这个阀值，类似于tableview的组头固定，可以引申。

z-index 堆叠元素，可为负数，默认应该是0吧，谁越大，就显示在上层。没有设置z-index,就看谁的代码写在后面就显示在上面，但是前面说的那几种定位就好像默认 都会提到最上面来的
